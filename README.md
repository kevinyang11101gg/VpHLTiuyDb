# Python教学评价管理系统简介

## 核心功能
- 用户管理
- 教学评价
- 数据统计
- 通知提醒
- 权限管理

## 适用场景
- 教学机构
- 教育管理平台
- 在线教育平台

## 技术特点
- 使用Python和Django进行后端开发
- 采用HTML、CSS、JavaScript、jQuery等技术实现前端界面
- 数据库使用MySQL
- 开发工具包括Pycharm和Navicat
- 系统兼容Python3.0以上版本和MySQL8.0以上版本
